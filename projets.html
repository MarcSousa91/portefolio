Application de gestion de chantiers MP Renov

Problème à résoudre. Centraliser les informations liées aux chantiers, utilisateurs, fichiers, images et données métier dans un outil exploitable par l'entreprise.

Migration Access vers PostgreSQL / outil de gestion de tickets

Problème à résoudre. Remplacer une base Microsoft Access locale, mono-utilisateur et fragile par un système centralisé, sécurisé et multi-utilisateurs.

Solution développée. Conversion .mdb avec mdbtools/MBTool, correction SQL/CSV, scripts Python UTF-8, base PostgreSQL avec triggers, logique PHP dans Scriptcase, RBAC, workflows, tableaux de bord, exports Excel et rapports PDF.

Impact métier. Données plus fiables, tickets mieux structurés, suivi SLA automatisé, historique d'audit et meilleure traçabilité des actions.

Application de gestion de budget communautaire

Problème à résoudre. Permettre à une communauté de communiquer, voter et prendre des décisions budgétaires collectives à partir de choix proposés.

Solution développée. API REST, base MySQL avec triggers et données fictives, site sécurisé avec page d'accueil et login, JavaScript, PHP, HTML/CSS, plus une application Java d'optimisation décisionnelle.

Impact métier. Centralisation des votes et aide à la décision pour retenir les choix les plus pertinents selon les préférences du groupe.

Rôle. Projet réalisé seul pour la partie web/base/API. L'application Java a été réalisée en équipe.

Problème à résoudre. Créer un jeu 3D rejouable avec des labyrinthes générés automatiquement et une difficulté modulable.

MEDIA4 Visualizer / simulateur de vitrine

Problème à résoudre. Aider un commercial à montrer rapidement à un client le rendu d'un écran publicitaire MEDIA4 dans une vitrine réelle.

Solution envisagée. Import d'une photo de vitrine, choix du type d'écran, placement par quatre coins avec perspective, ajout de photos produits, génération d'une publicité animée et export image ou MP4.

Impact visé. Rendre la proposition commerciale plus visuelle, plus rapide à partager par e-mail ou WhatsApp, et plus convaincante pour le client.
